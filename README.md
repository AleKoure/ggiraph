# ggiraph

> Make 'ggplot' Graphics Interactive
    
# Installation 

You will need the latest ggplot2 version and the latest version of 
rvg.


    install.packages("ggplot2")
    install.packages("rvg")
    devtools::install_github('davidgohel/ggiraph')
